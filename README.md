# 📊 PhonePe Transaction Analysis Dashboard

An interactive Power BI dashboard designed to analyze PhonePe transaction data and uncover valuable business insights related to customer behavior, transaction performance, service usage, and overall business growth.

---

## 📌 Project Overview

This project focuses on analyzing PhonePe transaction data using Power BI to monitor key business metrics and understand customer transaction patterns.

The dashboard provides a comprehensive view of:

- Transaction Performance
- Customer Activity
- Service Usage Trends
- Transaction Success Monitoring
- Business Growth Indicators

## 🔍 Key Insights

### 💰 Transaction Performance

- Total transaction value exceeded ₹3.47 Billion.
- More than 300K transactions were processed successfully.
- Monthly transaction growth remained positive throughout the year.

### 👥 User Insights

- Over 108K unique users were active on the platform.
- Millennials and Gen X contributed the largest share of transactions.

### 📱 Service Insights

- Loan services generated the highest transaction value.
- Insurance and Money Transfer services followed in contribution.

### 📅 Usage Insights

- Weekdays accounted for approximately 71.6% of total transactions.
- Weekend activity represented around 28.4% of transactions.

### ✅ Operational Insights

- Transaction success rate remained at 96%.
- The platform maintained strong transaction reliability.
